The Problem

For Organizers

  • • 90+ minutes manually managing brackets and pairings
  • • Paper scoring prone to errors and disputes
  • • No automated swiss pairings or tiebreakers
  • • Players constantly asking "When's my next match?"

For Players

  • • No visibility into rankings or next opponents
  • • Battle history scattered across notebooks and photos
  • • No way to track personal progress over time
  • • Hard to prove skill vs. luck without data

The Solution

One App. Two Roles. Zero Friction.

The Inertia app starts with one screen. Scan a QR code to check in as a player, or create a tournament to become an organizer. Same app, seamless experience, built for speed.

Organizer Features

  • ✓ Auto Swiss Pairings: Instant matchups with tiebreakers
  • ✓ Live Brackets: Players see updates in real-time
  • ✓ QR Check-In: No typing names, instant registration
  • ✓ Save 90 Minutes: Automated admin = more battles

Player Features

  • ✓ Match Tracking: See who you're fighting next
  • ✓ Battle History: Every match, every outcome, tracked
  • ✓ Performance Stats: Win rates, trends, insights
  • ✓ Build Proof: Data shows it's skill, not luck
